# Project Insights - DS108-Electrimight

This file summarizes the strongest evidence generated by the current pipeline.
It is written for academic review: the claims below are limited to values that
are present in the repository artifacts.

## Gold Dataset

The final dataset contains **35,040 rows and 69 columns**. It covers one full
year of 15-minute industrial electricity observations and combines source
electrical measurements, weather context, engineered features, and proxy anomaly
labels.

The aggregate proxy anomaly label `anomaly_any` is positive in **2,388 rows**,
equivalent to **6.815%** of the dataset.

## Insight 1: Temporal Features Drive Forecasting Performance

The forecasting task predicts `Usage_kWh(t+1h)`. In the current ablation study,
the best configuration by RMSE is **RAW + TIME**.

| Rank | Configuration | RMSE | MAE | R2 |
|---:|---|---:|---:|---:|
| 1 | RAW + TIME | 12.0087 | 6.1976 | 0.8535 |
| 2 | ALL ENGINEERED | 12.1154 | 6.3956 | 0.8509 |
| 3 | RAW + TIME + WEATHER + WAVELET | 12.1988 | 6.3676 | 0.8488 |
| 4 | RAW + TIME + WEATHER | 12.2585 | 6.4565 | 0.8473 |
| 5 | RAW + CONTEXT | 13.0119 | 6.7041 | 0.8280 |

Interpretation: recent consumption history and cyclic temporal structure explain
most of the short-term forecasting signal. Weather and wavelet features remain
useful for analysis, but they do not outperform the temporal feature set for the
current 1-hour forecasting target.

## Insight 2: Weather Context Improves Proxy Anomaly Prediction

For `anomaly_any` under the full-track cross-validation setting, the strongest
configuration by PR-AUC is **RAW + TIME + WEATHER**.

| Rank | Configuration | PR-AUC | F1 | Precision | Recall |
|---:|---|---:|---:|---:|---:|
| 1 | RAW + TIME + WEATHER | 0.3642 | 0.4232 | 0.5291 | 0.7214 |
| 2 | RAW + TIME + WEATHER + WAVELET | 0.3481 | 0.3851 | 0.4882 | 0.6310 |
| 3 | ALL ENGINEERED | 0.3432 | 0.3532 | 0.4612 | 0.5405 |
| 4 | RAW + CONTEXT | 0.1976 | 0.0250 | 0.0129 | 0.7500 |
| 5 | RAW + TIME | 0.1500 | 0.0796 | 0.0476 | 0.6143 |

Interpretation: weather variables are best presented as contextual enrichment
for proxy anomaly analysis. They should not be described as direct causal proof
of equipment faults.

## Insight 3: Rule-Free Results Define an Important Limitation

The rule-free track produces much lower PR-AUC values. The best rule-free
configuration reaches **0.0178 PR-AUC**, which shows that the proxy labels are
strongly tied to domain rules and engineered evidence.

| Rank | Configuration | PR-AUC | F1 | Precision | Recall |
|---:|---|---:|---:|---:|---:|
| 1 | ALL ENGINEERED | 0.0178 | 0.0006 | 0.0003 | 0.0571 |
| 2 | RAW + TIME + WEATHER + WAVELET | 0.0178 | 0.0006 | 0.0003 | 0.0571 |
| 3 | RAW + TIME + WEATHER | 0.0114 | 0.0006 | 0.0003 | 0.0571 |
| 4 | RAW + TIME | 0.0027 | 0.0022 | 0.0012 | 0.0571 |
| 5 | RAW + CONTEXT | 0.0012 | 0.0014 | 0.0007 | 0.3429 |

Interpretation: the project should not overclaim independent real-fault
detection. The stronger academic framing is that Electrimight builds an
auditable proxy-label benchmark and documents where domain assumptions matter.

## Insight 4: GAN Augmentation Is a Baseline, Not the Main Claim

The GAN experiment is useful as a baseline augmentation study for the minority
proxy anomaly class.

| Metric | Value |
|---|---:|
| Minority samples used for training | 2,388 |
| Synthetic samples generated | 500 |
| Mean error | 8.20% |
| Standard-deviation error | 3.81% |
| Correlation MAE | 0.116 |

Interpretation: the synthetic samples reproduce simple distributional summaries
reasonably well, but the correlation error remains material. GAN augmentation
should therefore be presented as an exploratory baseline, not as a replacement
for real industrial data.

## Final Reporting Position

The strongest defensible statement is:

> DS108-Electrimight turns raw industrial meter logs into an auditable analytical
> dataset. Temporal features provide the strongest 1-hour forecasting result,
> while weather context improves proxy anomaly prediction under the full-track
> ablation. The anomaly labels remain physics-informed proxies and require
> maintenance or SCADA evidence before being interpreted as real faults.
