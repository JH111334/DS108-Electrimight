# Danh Sách Tài Liệu Tham Khảo Gợi Ý — IEEE Format

> File này cung cấp các references đã format sẵn theo chuẩn IEEE. Bạn chỉ cần copy và sắp xếp lại theo **thứ tự xuất hiện** trong bài báo của mình.

---

## Nguyên Tắc Sắp Xếp References Trong IEEE

1. **Đánh số liên tục** `[1]`, `[2]`, `[3]`... theo thứ tự xuất hiện trong text
2. **Không** sắp xếp theo alphabet
3. Mỗi reference chỉ xuất hiện **một lần**
4. Nếu cần trích dẫn lại reference đã có, dùng lại số cũ

## Hướng Dẫn Trích Dẫn Trong Văn Bản

| Tình huống | Cách viết |
|-----------|-----------|
| Một reference | `...as shown in [3].` |
| Hai references liên tiếp | `...methods [4], [5] have...` |
| Nhiều references liên tiếp | `...studies [6]–[9] demonstrate...` |
| Reference là chủ ngữ | `Zhang et al. [6] proposed...` |
| Dataset | `The UCI dataset [1] contains...` |

---

> **Lưu ý quan trọng**: Số hiệu `[n]` trong file này chỉ là gợi ý. Khi viết báo cáo, bạn phải **đánh số lại** theo đúng thứ tự xuất hiện trong bài của mình.
